# Factory 11 Production-Line Illustration

Purpose: a visual explanation of the Overkill Factory as a product-production
line for autonomous agents.

## Generated Shot

16:9 editorial illustration, white background, hand-drawn black line art with a
small black operator character. Messy paper enters a factory machine on the
left. The machine passes through checkpoints:

```text
paper -> SOT -> architecture -> Product Face -> security -> review -> Receipt Five -> stable product
```

Small worker hands inspect screens, locks, code and onchain modules. A human
approval window appears near the Receipt Five gate. A stable product exits on
the right with a green status monitor.

Short annotations:

- `fonte`
- `gate`
- `prova`
- `humano`
- `risco`

## Use

Use this as the hero explanatory illustration for Factory 11 docs, not as a
technical diagram. The technical map remains `docs/maps/factory-10-flow.mmd`.

## Public Safety

The illustration must not include private product names, private board links,
local paths, real secrets, real wallet details, private runtime identifiers or
raw source screenshots.
