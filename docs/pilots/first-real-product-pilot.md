# First Real Product Pilot

## Current Baseline

A dry pilot already ran on a synthetic paper:

```text
pilot: pilots/quasar-vault-guard-test
Hermes board: overkill-factory-pilot-10
Hermes task: public-methodology-card
status: done
```

That dry pilot proved the factory process and Hermes gates. It did not prove a
real product, real Quasar code, real wallet behavior, devnet/mainnet behavior,
funds movement, custody, or production readiness.

## Objective

Run one real product paper through Overkill Factory from raw input to at least
one completed Hermes slice.

## Required Inputs

- Raw product paper.
- Any source material the product depends on.
- Human owner for decisions.
- Hermes board/workspace.

Use `templates/pilot-input-paper.md` if the paper needs a stable intake shape.

## Required Outputs

1. Source ledger.
2. Conflict matrix.
3. Product SOT.
4. Understanding questions and answer log.
5. Architecture candidate.
6. Product Face Packet.
7. Specialist review matrix.
8. Onchain Work Package when applicable.
9. Security Scan Packet.
10. Human architecture approval record.
11. Documentation OS.
12. Decomposition card graph.
13. At least one executed slice.
14. Receipt Five.
15. Independent review.
16. Pilot retrospective for V3.6.

## Prepared Templates

- `templates/pilot-input-paper.md`
- `templates/product-sot.md`
- `templates/architecture-review-packet.md`
- `templates/factory-card.json`
- `templates/receipt-five.json`
- `templates/human-gate-record.json`
- `templates/v3-6-learning-record.md`

## Pass Criteria

- No execution card bypasses ready gates.
- No done transition lacks Receipt Five.
- No required scan is represented by prose only.
- No R3/R4 approval is faked.
- The user can read the artifacts and understand the product path.

## Failure Criteria

- Agents ask broad business questions instead of execution-changing questions.
- Frontend/product face is postponed without an explicit gate.
- Onchain work is treated as generic backend.
- Security is summarized but not evidenced.
- Kanban state diverges from artifact truth.

## Current Blocker

This production-intent pilot cannot honestly run until a real product paper is
selected.
