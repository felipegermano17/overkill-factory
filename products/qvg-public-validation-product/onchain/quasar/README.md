# QVG Quasar Source

This is the Quasar source target for the public validation product. It is
intentionally small, but it is real source that must build, test and pass the
Auditor corpus/checklist contract before the production Auditor lane can be
marked achieved for this named product.

Boundaries:

- no deploy;
- no devnet or mainnet write;
- no signing authority;
- no funds movement;
- no claim that this validates any private product.
