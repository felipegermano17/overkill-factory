# QVG Public Validation Product

This directory contains the named public validation product used to test
production-scoped Overkill Factory lanes without using private product material.

It is not an internal product source tree. It is a public, reproducible target
for proving whether the factory can carry one product-shaped artifact through
strict Product Face, Quasar/Auditor, review and completion gates.

Production-lane evidence may be reusable only for the named product id:

`qvg-public-validation-product`
