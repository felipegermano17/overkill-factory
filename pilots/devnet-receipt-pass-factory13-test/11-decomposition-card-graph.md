  Decomposition Planner.
- Product: Product Face Validator, Implementation Worker, QA Verification.
- Security: Codex Security, AppSec OWASP, Agentic AI Security, Cloud/Infra,
  Crypto/Key Management, Supply Chain.
- Onchain: Solana/Quasar Auditor.
- Runtime: Remote Proof Runner.
- Review: AutoReview Gate, Independent Reviewer, Human Gate Clerk.
- Promotion: Release Operations, Detection and Monitoring, Public Safety.
- Continuity: Handoff Packer, Memory Steward, Skill Eval Distiller.

## Completion Rule

The card cannot close until before-done workers have valid result records and
the Receipt Five metadata validates. Before-ready workers are still generated
and recorded to prove the line can route the whole worker set.
